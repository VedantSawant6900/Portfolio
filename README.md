# Vedant Sawant Portfolio

Static personal portfolio for Vedant Sawant.

The site uses a dark glass-card layout, animated neural background, section navigation, experience timeline, skills tabs, project cards, and contact links. All public content is Vedant-specific and sourced from the portfolio research files in this repository.

## Run Locally

```bash
python3 -m http.server 3000
```

Open `http://localhost:3000`.

You can also open `index.html` directly in a browser.

## Content Files

- `portfolio_profile_research_questions.md`
- `portfolio_content.md`
- `portfolio_data.json`
- `asset_checklist.md`
- `implementation_notes.md`
- `verification_needed.md`

## Notes

- The public contact phone number and resume link are included by request.
- Exact address, visa details, internal healthcare data, private code, and unverified project claims are not published.
