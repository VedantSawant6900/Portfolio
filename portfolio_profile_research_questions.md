# Personal Portfolio Deep Research Questionnaire

Use this file as the source prompt for collecting everything needed to turn the Ateeb Shahid portfolio template into a portfolio for my profile. The goal is to remove all of Ateeb's identity, text, metadata, links, projects, images, and branding, then replace them with accurate, polished content about me.

Deep Research can be helpful if it has enough inputs: resume, LinkedIn, GitHub, project links, certificates, transcripts, job descriptions, screenshots, demos, and any public writing. It can help synthesize positioning, rewrite case studies, pull accomplishments from scattered sources, and recommend how to present my strongest story. It should not invent experience, dates, employers, metrics, credentials, or project impact.

## How To Answer

Answer every question with either:

- A direct answer
- A link to the source
- "Unknown"
- "Not applicable"
- "Do not include publicly"

When giving links, include the exact URL and whether it is public or private. When giving metrics, mention whether they are verified, estimated, or private.
